Designing a Website

Having the website blues? Your site styling burning everyone’s eyes? I personally ran into these issues since I am a programmer at heart. Not so much an artist or a designer. One day explaining my problems to a web designer he told me about a few good tools to make a guaranteed professional looking website. The first and most powerful tool is BootStrap. The main website has great tutorials on how to use the CSS code. If you are struggling with understanding how to do certain things on a website W3schools is a great source. W3schools explains all the resources needed to create and host a website. From making a professional basic splash page, to making it interactive, and finally hosting the page on a server. Even though W3schools gives you everything you need to build a website I personally find that I should flush out the basics for things like generic images.

Once getting the style, format, and layout of the site done many people turn their attention towards images. That is where Font Awesome comes into play. Font Awesome focuses on producing generic images that anyone can use. For instance if you are linking to twitter but want the twitter image to be easily scalable and a different color then you can add the Font Awesome reference and add the links to that image. The images are extremely easy to find by going to their website and using the search bar. Plus there are instructions to use all images for Font Awesome. We only have one glaring issue after making our webpage that is currently offline before we can/should bring it online. We need to add functionality.

If you get all the styling, images, and formatting done for your webpage you have made what designers call a splash page. It simply means the webpage shows information but can do nothing else. Well, what happens if you want to make a search bar for your site, or create a web builder, or a blog poster. Angular.js is one of many Java Scripting add-ons that makes coding a lot simpler. Javascript is what allows your website to actively get edited. That way you can make a format like the comments on a youtube video, but instead of custom coding every comment for every video Javascript can edit your code to not only add that comment into a file, but also reference that file and add all comments for the video while using your template. That way you can make a full working site that also changes itself to what the user desires.

Using all these resources should allow you to easily build a professional webpage that is fully customizable. Though there will always be the caveat that googling for a specific thing like a search bar in Angular.js, but at least now you know what to search for and the full basics to creating a webpage without needing to use a web builder. Which will allways be more customizable and cheaper if you do it yourself.
